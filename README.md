# Google analytics instructions

Go to analytics.google.com and create a free account
Set up a new property → choose Web → enter your website URL
You'll get a Measurement ID that looks like G-XXXXXXXXXX
Open the HTML file, find the two places that say G-XXXXXXXXXX and replace both with your real ID
That's it — data starts flowing immediately once the site is live
